# ポートフォリオサイトを作ってみました

「ポートフォリオサイトを作るぞ！」と宣言してから約１年が経ってしまいましたが、
今年の７月になってやっと作成に着手しはじめ、ついに完成させることができました。

本記事では、「何故ポートフォリオサイトを作ろうと考えた理由」や「作成に着手したきっかけ」など、
このポートフォリオサイト作成について触れたいと思います。

<br/>
# ポートフォリオサイトを作る理由

そもそも、自分がポートフォリオという概念を知ったのは、
好きなイラストレータさんがXfolioというポートフォリオサービスを利用しており、
そこで公開している作品を閲覧しに行ったことが切っ掛けです。

イラスト投稿サイトとは少し雰囲気が異なるXfolioを調べるうちに、
作品をまとめて不特定多数に公開する、ポートフォリオサイトという文化を知りました。

この時の自分は、IT業界でも同様にポートフォリオサイトを公開する文化があることを知らなかったのですが、
自分の制作物をポートフォリオサイトで紹介したいという気持ちを持ち始めました。

## Webの世界との邂逅

昨年度のはじめ、自分はIT技術を使い農家の人を助けられないか？と考えていました。
そこで、実行環境に左右されず、導入障壁が低いという理由からWebアプリの開発を始めました。

はじめてのWeb開発では、未知の概念が非常に多く開発は難航しましたが、
多くのことを学び成長できた開発経験だったと思います。
そして、Web技術をインプットした自分は、
#### 「ポートフォリオサイトを作りたい！」
と考えるようになりました。

## ポートフォリオサイトのメリット

さて、ポートフォリオサイトを作りたいというモチベーションは湧いてきましたが、
実際、ポートフォリオサイトを作る理由はあるのでしょうか？

もちろん個人の開発なので、損得は考えても仕方がないのですが、
学校や部活、他の私的な開発などの時間を割いて開発するとなると、
自分にとっては明確なメリットが欲しいところでした。

もったいぶらずに結論から言いますと、それらしいメリットはありました。  
（というか、なかったらこのサイトは存在しません）

- 技術的な勉強になる
- コンテストやカンファレンスに参加した際に使える
- 自分の実績のアピールになる
- 自分の経歴を見直すことができる

上記のようなメリットがあると、自分は考えました。

特に、技術的な勉強なるという点は自分にとって大きく、後述しますが、
ポートフォリオサイトを作る過程でフロントエンドの技術をインプットすることは、
作成に取り掛かる重要なファクターであり、ある意味での目的となります。

また、コンテストやカンファレンスなどのテック系イベントによく参加するのですが、
その際にポートフォリオサイトを紹介される方も多く、
「自分はこんな人ですよ」と簡単に紹介できる点も大きなメリットだと感じました。

<br/>
# 作成に着手した切っ掛け

先述したように、ポートフォリオサイトの作成に着手するまでに約１年間のラグがあります。
なぜ、すぐに着手しなかったのか、そして着手し始めた切っ掛けについて触れたいと思います。

## 空白の１年

１年前にポートフォリオサイトを作ろうと思い立った自分ですが、
その時期は絶賛「はたちゃ」の開発中でした。

「はたちゃ」とはWebの世界へ足を踏み入れる切っ掛けとなった上述のプロジェクトです。
ポートフォリオサイトを作る切っ掛けではありますが、最大の障壁でもあったのです。

ちなみに、「はたちゃ」の開発の終わりは明確に定義できませんが、現在は一段落ついた状態です。

ですが、「はたちゃ」の開発が一段落ついた直後に、
コンピュータフェスティバルのためにゲームの開発が始まってしまいました。

これによって、ポートフォリオサイトを作成するタイミングを逃してしまい、
ポートフォリオサイトへのモチベーションも過去のものになってしまったのです。

## 作成への着手

気がつけば４年生になり、就職活動に意識を向ける必要が出てきました。
そこで、４月からインターンシップを探しはじめ、
早期にSmartHRへのインターンシップが決定しました。

SmartHRでは、フロントエンドはNext.js、バックエンドはRubyを主に利用しているらしく、
インターンシップでもこれらの技術が必要となります。

しかし、当時の自分はフロントはVue.js、
バックエンドに関してはFirebaseやC#のASP.NETの経験しかなく、
未知の技術に触れる必要性が出てきたのです。

流石にこのままではマズイと考え、React.jsやRubyの学習をはじめました。
Rubyで簡単なAPIを作って遊んでみたり、FigmaでデザインしたWebサイトをReact.jsで作ってみたりして、
来る日へ備えていました。そんな最中、今にも頭のメモリからスワッピングされそうになっているポートフォリオサイトを参照しました。
そして、
#### ポートフォリオサイトを作りながら勉強すればええやん！！
という考えが舞い降りて来ました。

そして、React.jsやWeb知識の学習をすることを目的として、
ポートフォリオサイト作成を始めることになりました。
